# rui
React UI for our exploratory data api

![RUI Screenshot](../docs/frontend/rui.png)

## Development

Install dependencies:
```bash
npm install
```

Configure the database:
```bash
cp .env.example .env
```

Run the server:
```bash
npm run dev
```

Push to the gh-pages branch:
```bash
npm run deploy
```

Lint the code:
```bash
npm run lint
```

Fix lint errors:
```bash
npm run lint:fix
```

Format the code:
```bash
npm run format
```

Check formatting:
```bash
npm run format:check
```
