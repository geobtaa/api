# Service Tiers And Rate Limiting

The BTAA Geospatial API supports service tiers, API keys, and rate limiting.

Operational procedures for provisioning keys, changing deployed tier policies,
monitoring production usage, or troubleshooting deployed rate limiting are
restricted operations material.

Public contributors can inspect the implementation, tests, and API schemas in
the codebase. Do not publish production admin credentials, client keys,
customer-specific limits, or deployed monitoring procedures in public docs.
