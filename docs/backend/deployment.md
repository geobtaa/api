# Deployment

Detailed deployment and release operations are restricted to the private
operations documentation maintained by University of Minnesota Web Development.

This public page intentionally does not include deploy commands, host inventory,
SSH setup, secret-file names, bootstrap steps, rollback procedures, or
production troubleshooting commands.

Public contributors should use the local development, testing, architecture,
and API documentation in this repository. UMN Web Development deployers should
use the restricted operations runbook for release and production procedures.
